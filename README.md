# Instalador de Node Exporter para Prometheus

Script que instala y configura [Node Exporter](https://github.com/prometheus/node_exporter) en un servidor Linux, permitiendo que sea monitoreado por Prometheus.

## Uso

Accede al servidor que deseas monitorear y ejecuta:

```bash
curl -sSL https://gitlab.com/tu-usuario/infra-scripts/-/raw/main/install_node_exporter.sh | bash
```
